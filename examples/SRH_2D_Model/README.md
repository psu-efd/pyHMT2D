## Demonstrate the use of `SRH_2D_Model` class

This example demonstrates how to use **pyHMT2D**'s `SRH_2D_Model` class to control the run of SRH-2D.

```bash 
python demo_SRH_2D_Model.py
```

The python script will load the SRH-2D case file `Muncie.srhhydro`, run the case, and then convert the result to VTK.
