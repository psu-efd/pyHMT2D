# SRH-2D Model with SIF

This example demonstrates how to use *pyHMT2D*'s `SRH_2D_Model` class to control the run of SRH-2D using a SIF file. In addition to the option of using a srhhydro file (used by Aquaveo's SMS), the SIF file is another way to control the run of SRH-2D.

Note: This example is not fully tested. Some tweaks may be needed to make it work.
