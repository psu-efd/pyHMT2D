# Example 1: Open and Inspect a Project

This example demonstrates how to open and inspect a HEC-RAS 2D project using the AI Tools. For example, you can use the following prompt to get the information you need:

```text
I have a HEC-RAS 2D project in the directory Muncie. Please open the project, tell me what materials are defined and their Manning's n values, and list the boundary conditions.
```
