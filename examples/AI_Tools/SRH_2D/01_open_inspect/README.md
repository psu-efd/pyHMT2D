# Example 1: Open and Inspect a Project

This example demonstrates how to open and inspect an SRH-2D project using the AI Tools. For example, you can use the following prompt to get the information you need:

```text
I have an SRH-2D model in the current directory. The config file is ..\hmt_config.json and the control file is Muncie.srhhydro. Please load the config, open the project, tell me what materials are defined and their Manning's n values, and list the boundary conditions.
```
