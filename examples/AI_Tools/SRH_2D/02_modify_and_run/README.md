# Example 2: Modify and Run a Project

This example demonstrates how to modify and run an SRH-2D project using the AI Tools. For example, you can use the following prompt to get the information you need:

```text
The config file is ..\hmt_config.json and the SRH-2D model is Muncie.srhhydro in the current directory. Load the config, open the project, change Manning's n for the channel material to 0.035 and the left_2 material to 0.055, save the modified inputs, run preprocessing and simulation, then read the results and report the simulation status. Also report the maximum simulated water surface elevation.
```
