## Demonstrate the use of `HEC_RAS_Model` class

This example demonstrates how to use **pyHMT2D**'s `HEC_RAS_Model` class to control the run of HEC-RAS. 

```bash 
cd Muncie2D 
python demo_HEC_RAS_Model.py
```

The python script will load the HEC-RAS project file `Muncie2D.prj`, run the case, and then convert the result to VTK.
